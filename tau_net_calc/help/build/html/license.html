14. Appendix C: License¶

14.1. Calculation of the walking distance between two entities (buildings and/or stops)¶

The walking distance is computed during construction of the pkl dictionary.

  • Let {n} be the set of nodes of the road network G.

  • For each building and/or stop find the closest, by the aerial distance, node n.

  • Calculate network distance between two nodes that are closest to the entities, employing Dijkstra algorithm.

  • Total distance between two entities is calculated as the network distance plus two aerial distances between the entities and closest to them network nodes

14.2. How to exclude bus lines from the PT network¶

The list of lines to exclude is in the csv file exclude_routes.csv that has two fields seq_number, MAKAT. This file must be located in the Dataset folder folder

Example of the exclude_routes.csv file:

seq_number,MAKAT
1,79001
2,79002
3,79003

MAKAT must contain line’s מקט from the http://markav.net, see здесь.

14.3. How to add new bus lines to the PT network¶

The information about the new bus lines must be prepared in GTFS format and located in the Dataset folder in the subfolder add_routes.

The add_routes folder must contain GTFS files of the new lines: routes.txt, trips.txt, stops.txt, stop_times.txt, and the file of the roads in shp format

Note

In the GTFS files of the new lines:

  • routes’ catalog numbers must not repeat already existing ones.

  • stops’ codes must not repeat the codes of existing stops and buildings’ IDs

The dictionary’s addtion will be constructed in the pkl subfolder of the add_routes folder
