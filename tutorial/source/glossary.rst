Appendix A: Glossary
====================

.. glossary::

    GTFS
        `General Transit Feed Specification <https://developers.google.com/transit/gtfs>`_ is a common format for public transportation schedules and associated geographic information.
    
    Dijkstra Algorithm
        `Dijkstra's Algorithm <http://en.wikipedia.org/wiki/Dijkstra's_algorithm>`_ is an algorithm for finding the shortest paths between nodes in a graph, which may represent, for example, road networks.        

    pkl
        `Pickle <https://docs.python.org/3/library/pickle.html>`_, a Python module used for serializing and de-serializing a Python object structure.

    QGIS
        `QGIS <http://www.qgis.org>`_ is a free and open-source geographic information system application that supports viewing, editing, and analysis of geospatial data.

    RAPTOR Algorithm
        `RAPTOR Algorithm  <https://www.microsoft.com/en-us/research/wp-content/uploads/2012/01/raptor_alenex.pdf>`_ (Round-based Public Transit Routing) is algorithm for finding the optimal routes in public transportation networks.
